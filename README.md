This GitHub repository provides a computational framework for investigating the mechanical logic of the CRL4-CRBN E3 ubiquitin ligase complex through high-resolution structural analysis. By utilising Biopython to parse atomic coordinates from PDB entries such as 4OO2 and 6H0F, the project transforms static crystallographic data into dynamic insights regarding spatial orientation and target engagement. Central to the repository are Jupyter Notebooks that generate Alpha-Carbon Distance Matrices, allowing researchers to visualise distinct rectangular regions of domain folding and tertiary contacts that define the protein’s architecture. These tools are specifically designed to detect conformational tightening, providing a quantitative map of how molecular glue degraders stabilise the proximity between Cereblon and its substrates to facilitate efficient ubiquitination.
